sphinx-xref-example¶

This example provides an extension called mydoc.py which registers a custom directive and role.

  • mydoc: A role that works like the Sphinx doc role. This creates a hyperlink to another RST document.

  • mytoctree: A directive that works similar to the Sphinx toctree directive. This creates a bullet list of hyperlinks to RST documents
